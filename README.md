# llmasking-php

LLM-oriented PII masking and restoration library for PHP.

This is a PHP 8.1+ port of the Go `llmasking` core engine. It provides
identification, conflict resolution, strategy-based anonymization, and
session-based restoration of personally identifiable information (PII) in
free text.

## Status

Core engine under active implementation.
